WordPress到hexo的转变

两年以前，忽然想做一个个人博客，于是买域名，买服务器，买数据库（中国互联），安装WordPress，忙碌的一周就那么过去了，个人博客搭起来了，于是技术人生就开始了，不得不说，每个小白的内心都是梦想成为大神的。

今年4月的某一天，打开网址，忽然提示我This domain name expired on Mar 3 2017 12:51:54:000PM ，Click here to renew it.，瞬间懵逼，明明域名，服务器等都没到期，不知道要搞哪门子幺蛾，继续排查问题，原来php代码有问题，然后开始上网查看解决办法以及个人网站搭建的其他方法，看到hexo的瞬间，眼前一亮，作为一名前端，怎么可以不拥抱nodejs的产品呢。

hexo的搭建

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
github以及node的安装请参考官网https://hexo.io/zh-cn/docs/，安装好以后就可以将github和hexo关联起来了,

关联github

修改_config.yml中的deploy为

deploy: 
    type: git 
    repository: https://github.com/fay04/fay04.github.io.git 
    branch: master

将fay04替换成你的github用户名。

主题

设置主题https://hexo.io/themes/，hexo官网提供了n多主题，小伙伴们可以根据自己喜好去选择主题

在cmd中执行如下代码git clone https://github.com/klugjo/hexo-theme-phantom.git theme/strata,更新_config.yml中theme为strata，然后执行
hexo -g

hexo -d,

打开https://fay04.github.io/查看是否成功
